# OCR Text Extractor (Python)

A flexible and production-ready OCR utility for extracting text from images and PDF files using:

- OCR.space API (cloud-based)
- Tesseract OCR (local fallback)

Designed to be easily integrated into APIs, bots, or backend services.

---

## Features

- Image OCR via OCR.space API
- Local OCR fallback using Tesseract
- PDF to text extraction (multi-page)
- Automatic text chunking for long outputs
- Image validation and enhancement utilities
- Language support (Arabic & English)
- Clean, reusable Python module
